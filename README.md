# Movie-Recommender

The datasets for this project is downloaded from the top right section of this webpage https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata?select=tmdb_5000_movies.csv


This is an end to end content based recommender system built using cosine similarity 


Tech Stack Used : Python, Scikit-Learn, Numpy, NLTK, streamlit
